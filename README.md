# OpenCV를 활용한 간단한 video recorder

- ### 실행 예시

<img src="./실행 예시.png" alt="image1"/>

- ### 모드

n(normal) : 기본 모드

r(recording) : 녹화 모드(녹화 모드 진입 시 오른쪽 밑에 보라색 원 생성)

c(capture) : 이미지 캡쳐

space : 기본 모드와 녹화 모드로 번갈아가며 바꿈

ESC : 즉시 종료
